# laravel-bank
Aplicacion que simula la banca electronica de un cliente del banco.

# Funcionalidades
- Alta de clientes
- Transferencias de saldo entre clientes del banco.
- Creacion de cuentas adicionales

# Tecnologias
- Laravel
- Vue/Vuex/VueRouter
- Bootstrap/BootstrapVue
- Laravel Echo (websocket)
- AdminLTE???
